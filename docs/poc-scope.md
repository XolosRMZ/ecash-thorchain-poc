# PoC Scope

## Goal
Explore how Tonalli Wallet could support a future eCash interoperability path with THORChain at the wallet and UX layers.

## In Scope
- THORChain API connectivity
- inbound address discovery
- swap quote retrieval
- memo flow research
- wallet-side architecture
- conceptual UI and transaction orchestration planning

## Out of Scope
- native XEC support inside THORChain
- production-safe fund routing from XEC into THORChain
- Bifrost implementation for eCash
- mainnet claims of interoperability

## Workstreams

### Workstream A — Wallet-side PoC
- quote service
- inbound address resolver
- memo inspection
- conceptual flow UI

### Workstream B — Chain-level research
- eCash chain-client requirements
- UTXO compatibility analysis
- Bifrost integration gap analysis
- governance and rollout considerations
